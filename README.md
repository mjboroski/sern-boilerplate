# SERN Boilerplate

Simple Full-Stack Javascript boilerplate for Express, Sequilize, Postgresql, React, and Node based applications.

### Technologies Used

  * Axios
  * NodeJS
  * NPM
  * Express
  * Webpack
  * Babel/ES6
  * React
  * Redux
  * Passport
  * Nodemon
  * Sequelize
  * Sequelize-CLI
  * PostgreSQL
  
### Getting Started

```
npm install
npm start
````
